发表于 2016-11-09   |     |   阅读次数

引言

在搭建成功Ceph集群后，对于如何使用，其实我还是一脸MB的，心想竟然提供三种存储接口（对象，文件，快），口气也未免太大。在结合项目需求之后，我选择了对象存储接口。那么问题又来了，如何配置IPv6的对象存储？

实验环境

  • Linux操作系统版本：CentOS Linux release 7.2.1511 (Core)
    Minimal镜像 603M左右
    Everything镜像 7.2G左右
  • Ceph版本：0.94.9（hammer版本）
